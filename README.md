A self-hostable bot for playing youtube videos in discord voice channels. *Try it out!*
Latest stable python version as of 10/12/2025 (DD/MM/YYYY) recommended.
You should create your own .env with "DISCORD_TOKEN=...". (How to find your discord token: youtube.com/watch?v=aWtc_kpqSVk)
All requirements to run the bot are listed in requirements.txt!
